# RP5 weather export — бесплатная версия для Render Free

Это облегченная stateless-версия сервиса под бесплатный Render.

## Что умеет
- Показывает простой русскоязычный интерфейс.
- Принимает CSV-файл со списком городов.
- Формирует один Excel-файл на лету.
- Отдает Excel сразу в браузер на скачивание.
- Не требует базы данных и persistent disk.

## Почему эта версия подходит для Render Free
У бесплатного Render нет надежного постоянного хранения файлов между перезапусками и деплоями. Поэтому этот вариант не хранит историю запусков и не держит список городов в SQLite. Вместо этого пользователь загружает CSV на каждый запуск, а результат сразу скачивает себе на компьютер.

## Структура CSV
Файл должен содержать колонки:
- `name`
- `rp5_url`
- `sheet_name`

Пример лежит здесь:
- `app/static/cities_example.csv`
- `sample_input/cities_example.csv`

## Что сейчас важно
В `app/parser.py` стоит временная заглушка. Она нужна, чтобы:
- интерфейс уже можно было задеплоить и проверить;
- Excel-структуру уже можно было проверить;
- на следующем этапе заменить заглушку на реальный парсер/загрузчик RP5.

## Как задеплоить на Render
1. Создай репозиторий на GitHub.
2. Загрузи туда все файлы проекта.
3. В Render нажми `New` → `Blueprint`.
4. Подключи репозиторий.
5. Render прочитает `render.yaml` и создаст бесплатный web service.
6. После деплоя открой URL сервиса.

## Локальный запуск
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --reload
```

## Что будет в Excel
- Лист `README`
- Лист `ALL_DATA`
- Отдельные листы по каждому городу

## Следующий шаг
После деплоя нужно заменить заглушку в `app/parser.py` на реальный механизм получения архива RP5.
