# RP5 weather export — финальная бесплатная версия для Render Free

Это бесплатная stateless-версия сервиса для Render Free.

## Что делает сервис
- Принимает CSV со столбцами `name` и `sheet_name`.
- Сам пытается найти архивную страницу RP5 по названию города.
- Получает station_id через пакет `weather-rp5`.
- Загружает архивные данные.
- Собирает один Excel-файл с листом `ALL_DATA` и отдельными листами по городам.

## Важно
- Если сервис не может уверенно выбрать станцию, в Excel появится `match_status = ambiguous`.
- Если RP5 изменит внутреннюю логику страниц, поиск может потребовать доработки.
- На бесплатном Render сервис не хранит историю запусков и файлы постоянно.

## Структура CSV
Файл должен содержать:
- `name`
- `sheet_name`

## Как задеплоить на Render
1. Создай репозиторий на GitHub.
2. Загрузи туда все файлы проекта.
3. В Render нажми `New` → `Blueprint`.
4. Подключи репозиторий.
5. Render создаст бесплатный web service по `render.yaml`.
6. После деплоя открой URL сервиса.
